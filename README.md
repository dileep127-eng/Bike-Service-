# Bike Service Android App

## Project Overview

The Bike Service Android app is designed to assist motorcycle owners in managing their servicing needs effectively. The application tracks various aspects of bike maintenance, including:

- Servicing reminders
- Odometer readings
- Management of spare parts
- Tracking fuel expenses
- Exporting data to Excel with Google Drive integration

This app aims to ensure that users can stay on top of their bike servicing requirements and expenses, helping to prolong the life of their vehicles and improve efficiency.

## Features

- **Servicing Reminders:** Set and receive notifications for upcoming service intervals.
- **Odometer Tracking:** Log odometer readings to monitor bike usage and schedule maintenance accordingly.
- **Spare Parts Management:** Keep track of spare parts used and needed for maintenance.
- **Fuel Expenses Tracker:** Record fuel expenses and analyze consumption over time.
- **Data Export:** Export your data to Excel format and integrate with Google Drive for easy access and sharing.

## Getting Started

### Installation
1. Clone the repository using `git clone <repository-url>`.
2. Open the project in Android Studio.
3. Build the project and run it on your Android device or emulator.

### Usage
- Launch the app to start tracking your bike servicing needs.
- Access different features from the main dashboard.
